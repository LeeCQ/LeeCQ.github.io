2. 在- (BOOL)application:(UIApplication )application didFinishLaunchingWithOptions:(NSDictionary )launchOptions方法中注册

3. 重载方法

这个方法主要是判断是否要拦截请求，返回NO则不做处理。可以根据自己的需求做相应的逻辑判断。

此方法可以对request做一些处理，一般直接返回。

前面 canInitWithRequest 方法中返回YES，拦截到请求则会调用startLoading方法，这里就可以大作文章了。但这里也必须将拦截到的请求重新出去，否则拦截到的请求将不会继续。

还要就是可以通过[[self class] setProperty:@YES forKey:kFiltter inRequest:mutableRequest]; 给请求添加一个flag 并在+canInitWithRequest 中做判断,以防死循环
